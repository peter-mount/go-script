if is a statement that allows for code to be executed based on the result of an expression.

Syntax

if expression trueStatement
if expression trueStatement else falseStatement
The expression is true when the result of the expression is either a boolean true, a non 0 numerical value or one of the following strings: "true", "t", "yes" or "y"

When the if statement is encountered, the expression is evaluated and if the result is true then the trueStatement is executed. If the result is false and the else clause is present then the falseStatement is executed.

As all statements, multiple statements can be presented when wrapped within a { } pair.

For example:

if a > b fmt.Println("a > b") if a > b { fmt.Println("a > b") } if a>b { fmt.Println("a > b") } else { fmt.Println("a <= b") }
