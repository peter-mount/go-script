--- type: "manual" title: "Do" titleClass: section linkTitle: "Do" description: "repeat a loop whilst an expression is true" tags: - go-script ---

do allows for a statement to be executed whilst an expression returns true.

Unlike while, do will always execute the statement at least once as the expression is only evaluated after the statement has been executed.

Syntax

do statement while expression
The expression is true when the result of the expression is either a boolean true, a non 0 numerical value or one of the following strings: "true", "t", "yes" or "y"

When the do statement is encountered, the statement is evaluated followed by the expression. If the expression returns true then the statement is repeated.

The statement is always executed once.

Examples

The following code will print out the numbers 10 down to 3:

a:=10 b:=3 do { fmt.Println(a) a = a - 1 } while a >= b
