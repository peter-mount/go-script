--- type: "manual" title: "{ }" titleClass: section linkTitle: "{ }" description: "Statements wrapped within { }" tags: - go-script ---

Wherever a statement is expected, you can place a set of statements wrapped within a pair of { }.

This is useful in many ways, mainly within loops, conditionals etc. where you need multiple statements, or to define the scope of variables.

When it comes to variables, any variable that is declared within a { } block will only be visible within that block. When the block terminates, those variables are lost.
