while allows for a statement to be executed whilst an expression returns true.

Unlike repeat, while will not execute the statement at all if the expression is true when the statement first executes.

Syntax

while expression statement
The expression is true when the result of the expression is either a boolean true, a non 0 numerical value or one of the following strings: "true", "t", "yes" or "y"

When the while statement is encountered, the expression is evaluated and, whilst the result is true will execute the statement.

If the expression never returns true then the statement will never be executed.

Examples

The following code will print out the numbers 10 down to 3:

a:=10 b:=3 while a > b { fmt.Println(a) a = a - 1 }
