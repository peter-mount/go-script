switch allows for a statement to be executed if it's associated expression either equals the initial expression or, if the initial expression is not set returns true.

Syntax

switch expression {
    case expression : statement
    case expression : statement
    …
    default: statement
}
switch {
    case expression : statement
    case expression : statement
    …
    default: statement
}

Case selection

A switch statement can contain multiple case entries, one for each possible match. The order of each case is important as the first one to match will run.

Unlike in C but like Go, execution does not fall through to the next case block. When a case is executed the switch statement terminates.

When the switch expression { } form is used, the expression is evaluated and the result of each case expression must equal that value to be invoked.

The expression is true when the result of the expression is either a boolean true, a non 0 numerical value or one of the following strings: "true", "t", "yes" or "y"

If the initial expression is not declared then the first case expression which returns a true result will be executed.

Multiple expressions per case

If a case can match multiple expressions, you can indicate this by listing each expression as a , separated list.

Default selection

A switch statement may include an optional default statement. If present and no case matches, then this code will instead be executed. The default statement must be declared after all case clauses within the switch statement.

Examples

The following two examples are functionally identical:

a:=10 switch a { case 1: fmt.Println(1) case 2: fmt.Println(2) case 10: { fmt.Println("Found it!") } default: fmt.Println("No case matched") }
a:=10 switch { case a==1: fmt.Println(1) case a==2: fmt.Println(1) case a==10: { fmt.Println("Found it!") } default: fmt.Println("No case matched") }

Here we have a single case with three expressions: 10, 11 and 12. When the variable a matches any of those then the same statement is executed.

for a:=10; a<13; a=a+1 { switch a { case 1: fmt.Println(1) case 2: fmt.Println(2) case 10, 11, 12: { fmt.Println("Found it!") } default: fmt.Println("No case matched") } }
