--- type: "manual" title: "Return" titleClass: section linkTitle: "Return" description: "Return from the current function" tags: - go-script ---

return is a statement that will immediately exit the current function, optionally returning a single value to the caller.

Internally, return is implemented as an error. However, it is not possible to catch this error within a try catch statement.

Syntax

return
return expression

Examples

main() { // No return value example1() // Call a function and print it's result fmt.Println( example2() ) // Call the same function ignoring the result example2() } // example1 returns nothing example1() { for a := 1; a < 10; a = a + 1 { if a == 5 { return } } } // example2 returns 5 example2() { for a := 1; a < 10; a = a + 1 { if a == 5 { return a } } // never reached return -1 }
