--- type: "manual" title: "Break" titleClass: section linkTitle: "Break" description: "Break out of the current loop" tags: - go-script ---

break is a statement that will break out of the current for, repeat or while loop.

A break outside of a loop will generate an error.

Internally, break is implemented as an error. However, it is not possible to catch this error within a try catch statement.

Syntax

break

Examples

for i:=1; i<=10; i=i+1 { fmt.Println(a) if i==5 break }
