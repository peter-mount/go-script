--- type: "manual" title: "Repeat" titleClass: section linkTitle: "Repeat" description: "repeat a loop until an expression is true" tags: - go-script ---

repeat allows for a statement to be executed until an expression returns true.

Unlike while, repeat will always execute the statement at least once as the expression is only evaluated after the statement has been executed.

Syntax

repeat statement until expression
The expression is true when the result of the expression is either a boolean true, a non 0 numerical value or one of the following strings: "true", "t", "yes" or "y"

When the repeat statement is encountered, the statement is evaluated followed by the expression. If the expression returns true then the statement is repeated.

The statement is always executed once.

Examples

The following code will print out the numbers 10 down to 3:

a:=10 b:=3 repeat { fmt.Println(a) a = a - 1 } until a < b
