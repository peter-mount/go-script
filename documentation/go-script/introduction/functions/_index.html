In go-script a function declares a reusable block of code which takes a fixed number of parameters and can optionally return a value.

Declaring a function

In a script you simply declare a function by specifying its name, any parameters within parentheses and a block of statements.

myfunction() { // code goes here }
These variables are only accessible from within the body of the function.
They cannot be accessed outside the function, nor from any function that is called by this function.

Parameters are placed within the parentheses. Because the language is not type safe you only need to declare the variable names they will be referenced by within the function.

myfunction( param1, param2 ) { // code goes here }

Returning values

A function can return a single value via the return statement.

myfunction( param1, param2 ) { return param1 }

Because return only allows a single value, to return more than one value you need to use either an array or a map.

Local functions

Function names are global in scope, so if you include another script via the include statement then it's functions are available to your script.

To allow for a script to create functions only it will be able to call we have the concept of local functions. These are functions whose name begins with a _ character. These functions are only callable from within the script they are defined in.

The main function

If you want your script to be executable from the command line you need to declare the main() function.

It takes no parameters and is not expected to have a value returned from it. With this function present you can call the script from your application.

main() is special in that it is local to the script it was defined in. This allows for scripts to have their own main() function which will be ignored if those scripts are then included from another script.
